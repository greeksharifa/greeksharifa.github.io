스택은 매우 유용하다. 여러분의 실생활에서도 자주 볼 수 있다. 그리고 알고리즘으로 적용하기에도 쉽다. 그럼 스택은 무엇인가?

### Stack (스택)

스택([stack](https://en.wikipedia.org/wiki/Stack_(abstract_data_type)))의 사전적 의미는 더미(무더기)이다.

```python
print(1)
c = input()
print(2*c + 1)
def pp(c):
    print('what?')
    print(c)
pp(c+5)
```
